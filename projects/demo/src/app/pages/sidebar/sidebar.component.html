Sidebar Component

Componente moderno de barra lateral con Signals y animaciones CSS

Ejemplos Interactivos

Sidebar Izquierdo

Abre el sidebar desde el lado izquierdo de la pantalla.

Estado: {{ leftSidebarOpen() ? 'Abierto' : 'Cerrado' }}

Sidebar Derecho

Abre el sidebar desde el lado derecho de la pantalla.

Estado: {{ rightSidebarOpen() ? 'Abierto' : 'Cerrado' }}

Características

Basado en Signals

Utiliza la nueva API de Signals de Angular 20+

Animaciones CSS

Transiciones suaves sin Angular Animations

Posicionamiento

Configurable a izquierda o derecha

Responsive

Adaptado para dispositivos móviles

Ejemplo de Uso

<!-- En tu componente -->
<tng-sidebar 
  #mySidebar
  [position]="'left'"
  [width]="280"
  [showOverlay]="true"
  (opened)="onSidebarOpened()"
  (closed)="onSidebarClosed()"
>
  <h2>Contenido del Sidebar</h2>
  <p>Aquí va tu contenido...</p>
</tng-sidebar>

<!-- Botón para abrir -->
<button (click)="mySidebar.open()">
  Abrir Sidebar
</button>
